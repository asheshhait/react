C Model Architecture

          

            
🔍 Detection: YOLOv11

            
• Input: 640×640 RGB

            
• Backbone: CSPDarknet

            
• Head: PANet + YOLO Head

            
• NMS threshold: 0.45

            
• Conf threshold: 0.25

            

            
🧠 Recognition: CRNN

            
• CNN: VGG-like (7 conv layers)

            
• RNN: 2-layer Bi-LSTM (256 units)

            
• Decoder: CTC Loss + Beam Search

            
• Vocab: 252 Bengali graphemes

• Δx, Δy coordinates

            
• Timestamp delta

            
• Velocity (px/ms)

            
• Curvature angle

            
• Direction (8-way)

            
• Pen up/down flag

            
• Stroke length & duration
